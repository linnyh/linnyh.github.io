дё»иҰҒиҙЎзҢ®

  • дёәдәҶжңүж•Ҳең°зј–з ҒеҜ№иҜқдёҠдёӢж–ҮпјҢжң¬ж–ҮеҲ©з”Ёд»ҘеүҚзҡ„еҜ№иҜқзҠ¶жҖҒпјҲйў„жөӢзҡ„пјүе’ҢеҪ“еүҚзҡ„еҜ№иҜқиҜқиҜӯдҪңдёәDSTзҡ„иҫ“е…ҘгҖӮ

й—®йўҳе®ҡд№ү

вҖӢ еҒҮи®ҫеӯҳеңЁ $M$ дёӘеҹҹпјҢ$\mathcal{D}={ d1,d_2,вҖҰ,d_M }$, жҜҸдёҖдёӘеҹҹd $d \in \mathcal{D}$ зҡ„ж§ҪйӣҶеҗҲдёә$\mathcal{S}^d = { s_1^d,s_2^d,вҖҰ,s^d{|\mathcal{S}^d|} }$ еӣ жӯӨпјҢе®Ңе…ЁеҸҜиғҪеӯҳеңЁ $J = \sum_{m=1}^M |\mathcal{S}^{d_m}|$ дёӘеҸҜиғҪзҡ„ еҹҹ-ж§Ҫ еҜ№ $\mathcal{O} = { O_1,O_2,вҖҰ,O_J }$гҖӮз”ұдәҺдёҚеҗҢзҡ„еҹҹеҸҜиғҪеҢ…еҗ«зӣёеҗҢзҡ„ж§ҪпјҢеӣ жӯӨжҲ‘д»¬иЎЁзӨәжүҖжңүдёҚеҗҢзҡ„$N$дёӘжҸ’ж§ҪпјҢеҰӮ$S={s_1гҖҒs_2гҖҒВ·В·В·гҖҒS_N }$пјҢе…¶дёӯ$NвүӨJ$гҖӮ

вҖӢ еҜ№иҜқеҸҜд»ҘжӯЈејҸиЎЁзӨәдёә е…¶дёӯ $At$ дёәз¬¬$t$иҪ®зҡ„зі»з»ҹиҜқиҜӯпјҢ$U_t$ дёәз”ЁжҲ·иҜқиҜӯпјҢ$B_t$иЎЁзӨәзӣёеә”зҡ„еҜ№иҜқзҠ¶жҖҒгҖӮ$A_t$е’Ң$U_t$жҳҜеҚ•иҜҚеәҸеҲ—пјҢиҖҢ$B_t$жҳҜдёҖз»„еҹҹж§ҪеҖјдёүе…ғз»„пјҢдҫӢеҰӮ ,еҖј$v{tj}$жҳҜз¬¬ t еӣһеҗҲз¬¬ j еҹҹж§ҪеҜ№зҡ„еҚ•иҜҚеәҸеҲ—гҖӮDSTзҡ„зӣ®ж ҮжҳҜж №жҚ®еҜ№иҜқеҺҶеҸІпјҢжӯЈзЎ®ең°йў„жөӢжҜҸдёӘеҹҹ-жҸ’ж§ҪеҜ№зҡ„еҖјгҖӮ

вҖӢ д№ӢеүҚзҡ„еӨ§еӨҡж•°и®әж–ҮйғҪйҖүжӢ©иҝһжҺҘеҜ№иҜқеҺҶеҸІдёӯзҡ„жүҖжңүеҚ•иҜҚпјҢ$[A_1гҖҒU_1гҖҒA_2гҖҒU_2гҖҒВ·В·В·гҖҒA_tгҖҒU_t]$дҪңдёәиҫ“е…ҘгҖӮз„¶иҖҢпјҢиҝҷеҸҜиғҪдјҡеҜјиҮҙи®Ўз®—ж—¶й—ҙзҡ„еўһеҠ гҖӮеңЁжң¬ж–ҮдёӯпјҢд»…еҲ©з”ЁеҪ“еүҚзҡ„еҜ№иҜқеӣһеҗҲ$AtгҖҒU_t$е’Ңд№ӢеүҚзҡ„еҜ№иҜқзҠ¶жҖҒ$B{tвҲ’1}$жқҘйў„жөӢж–°зҡ„зҠ¶жҖҒ$Bt$гҖӮеңЁи®ӯз»ғиҝҮзЁӢдёӯпјҢеҲҷжҳҜдҪҝз”ЁдәҶ$B{tвҲ’1}$зҡ„зңҹе®һеҖјпјҢиҖҢд№ӢеүҚйў„жөӢзҡ„еҜ№иҜқзҠ¶жҖҒе°Ҷиў«з”ЁдәҺжҺЁзҗҶйҳ¶ж®өгҖӮ

Schema Graphпјҡ дёәдәҶиҖғиҷ‘дёҚеҗҢ domain-slot еҜ№д№Ӣй—ҙзҡ„е…ізі»пјҢе№¶дё”еҲ©з”Ёе®ғд»¬дҪңдёәдёҖдёӘйўқеӨ–зҡ„иҫ“е…ҘпјҲе…ҲйӘҢзҹҘиҜҶпјүжқҘжҢҮеҜјдёҠдёӢж–Үзј–з ҒпјҢжң¬ж–Үе°Ҷе®ғд»¬иЎЁзӨәдёәдёҖдёӘжЁЎејҸеӣҫ$G=(VпјҢE)$пјҢе…·жңүиҠӮзӮ№йӣҶVе’Ңиҫ№зјҳйӣҶEгҖӮеҰӮдёӢеӣҫпјҡ

еңЁеӣҫдёӯпјҢжңүдёүз§ҚиҠӮзӮ№еҲҶеҲ«иЎЁзӨәжүҖжңүеҹҹ$\mathcal{D}$гҖҒж§Ҫ$\mathcal{S}$е’Ңеҹҹ-ж§ҪеҜ№$\mathcal{O}$пјҢеҚі$V=DвҲӘSвҲӘO$гҖӮеҲ©з”ЁдёҚеҗҢиҠӮзӮ№д№Ӣй—ҙзҡ„еӣӣз§Қж— еҗ‘иҫ№жқҘзј–з Ғе…ҲйӘҢзҹҘиҜҶпјҡ

  • $(d,d^{вҖҳ})$: дёҚеҗҢеҹҹд№Ӣй—ҙзҡ„иҫ№
  • $(s,d)$: еҹҹдёҺж§Ҫд№Ӣй—ҙзҡ„иҫ№пјҢеҸӘжңүеңЁ$d\in \mathcal{D} дё” s\in \mathcal{S}^d$ ж—¶жңүй“ҫжҺҘ
  • $(d,o) е’Ң (s,o)$: еҰӮжһң domain-slot еҜ№ $o \in \mathcal{O}$ жҳҜз”ұ $d\in \mathcal{D} е’Ң s\in \mathcal{S}$ з»„жҲҗпјҢйӮЈд№Ҳе°ұжңүдёӨжқЎиҫ№еҲҶеҲ«д»Һ $d$ е’Ң $s$ иҝһжҺҘпјҲиҷҡзәҝпјүеҲ°иҝҷдёӘеҹҹ-жҸ’ж§ҪиҠӮзӮ№
  • $(s,sвҖҷ)$: еҰӮжһңдёӨдёӘдёҚеҗҢжҸ’ж§ҪпјҲ$s \in \mathcal{S} дё” s^{вҖҳ} \in \mathcal{S}$пјүзҡ„еҖҷйҖүеҖјйҮҚеҸ пјҢе®ғд»¬д№Ӣй—ҙд№ҹжңүдёҖдёӘиҫ№зјҳпјҢдҫӢеҰӮпјҢзӣ®зҡ„ең°е’ҢзҰ»ејҖгҖҒзҰ»ејҖе’ҢеҲ°иҫҫгҖӮ

  • Previous Dialog State

  • Schema Graph

    еҰӮеүҚжүҖиҝ°пјҢжЁЎејҸеӣҫGз”ұMдёӘеҹҹиҠӮзӮ№гҖҒNдёӘж§ҪиҠӮзӮ№е’ҢJдёӘеҹҹ-ж§ҪиҠӮзӮ№з»„жҲҗгҖӮиҝҷдәӣиҠӮзӮ№зҡ„жҺ’еҲ—ж–№ејҸдёәпјҡ

    жҜҸдёӘиҠӮзӮ№зҡ„еөҢе…ҘйғҪжҳҜйҖҡиҝҮе№іеқҮзӣёеә”еҹҹ/жҸ’ж§Ҫ/еҹҹжҸ’ж§Ҫдёӯзҡ„tokenеөҢе…ҘжқҘеҲқе§ӢеҢ–зҡ„гҖӮеңЁеӣҫдёӯзңҒз•ҘдәҶдҪҚзҪ®еөҢе…ҘгҖӮеӣҫзҡ„иҫ№иў«иЎЁзӨәдёәдёҖдёӘйӮ»жҺҘзҹ©йҳө$A^G$пјҢе…¶йЎ№иҰҒд№ҲжҳҜ1иҰҒд№ҲжҳҜ0пјҢиҝҷе°ҶеңЁиһҚеҗҲзҪ‘з»ңдёӯдҪҝз”ЁгҖӮдёәдәҶејәи°ғдёҚеҗҢзұ»еһӢиҠӮзӮ№д№Ӣй—ҙзҡ„иҫ№еңЁи®Ўз®—дёӯеҸҜиғҪдјҡжңүжүҖдёҚеҗҢпјҢжң¬ж–ҮеҲ©з”ЁиҠӮзӮ№зұ»еһӢжқҘиҺ·еҫ—ж®өеөҢе…ҘгҖӮ

Context and Schema Fusion NetworkпјҲCSFNпјү

вҖӢ еҲ©з”ЁдёҠдёӢж–Үе’ҢжЁЎејҸиһҚеҗҲзҪ‘з»ң(CSFN)йҖҗеұӮи®Ўз®—$XtгҖҒB{tвҲ’1}$е’Ң $G$ дёӯзҡ„tokenжҲ–иҠӮзӮ№зҡ„йҡҗи—ҸзҠ¶жҖҒгҖӮиҜҘзҪ‘з»ңжҳҜз”ұ $L$ еұӮ дёҠдёӢж–ҮжЁЎејҸж„ҹзҹҘиҮӘжіЁж„ҸеҠӣеұӮе ҶеҸ иҖҢжҲҗпјҢдёӢйқўе°ҶиҜҰз»Ҷд»Ӣз»Қе…¶з»“жһ„пјҡ

  • еӨҡеӨҙжіЁж„ҸеҠӣжЁЎеқ—

    иҖғиҷ‘еҗ‘йҮҸеәҸеҲ— $Y \in R^{|Y| \times d{model}}$, д»ҘеҸҠзӣ®ж ҮеәҸеҲ— $Z \in R^{|Z| \times d{model}}$, еҜ№дәҺ Y дёӯзҡ„жҜҸдёҖдёӘtoken $y_i$, еҲ©з”ЁHдёӘжіЁж„ҸеҠӣеӨҙи®Ўз®—е…¶еңЁ Z дёҠзҡ„жіЁж„ҸеҠӣеҗ‘йҮҸ $c_i$ пјҡ

    е…¶дёӯ е№¶дё”

    жҲ‘д»¬еҸҜд»Ҙи®Ўз®—жҜҸдёӘ$yi$зҡ„$c_i$пјҢе№¶еҫ—еҲ°дёҖдёӘеҸҳжҚўзҹ©йҳө$CвҲҲR^{|Y|\times d{model}}$ж•°жҚ®жЁЎеһӢгҖӮж•ҙдёӘиҝҮзЁӢеқҮиЎЁзӨәдёәдёҖдёӘжҳ е°„$MultiHead_Оҳ$пјҡ

  • еҹәдәҺеӣҫзҡ„еӨҡеӨҙжіЁж„ҸеҠӣжЁЎеқ—

    дёәдәҶе°ҶеӨҡеӨҙжіЁж„ҸеҠӣеә”з”ЁеҲ°еӣҫдёҠпјҢеӣҫйӮ»жҺҘзҹ©йҳө$AвҲҲR^{|Y|Г—|Z|}$ж¶үеҸҠеҲ°жҺ©и”Ҫж— е…ізҡ„иҠӮзӮ№/д»ЈзүҢпјҢе…¶дёӯ$A{ij}вҲҲ{0,1}$гҖӮеӣ жӯӨпјҢ$e^{(h)}{ij}$иў«еҸҳжӣҙдёәпјҡ
